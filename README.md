## ğŸ” Features

- **User Authentication:** Register and login with OTP-based mobile verification
- **Protected Routes:** Dashboard only accessible with valid authentication
- **Restaurant Discovery:** Browse and explore restaurants with detailed information
- **Restaurant Details:** View comprehensive restaurant cards with images, ratings, and descriptions
- **Personalized Recommendations:** "Your taste" section with tailored restaurant suggestions
